# Pac-Man CLI Arcade+

A colorful arcade-style Pac-Man game for the terminal, written in pure Python using the standard-library `curses` module.

## Features

- Real-time keyboard movement with arrow keys or WASD
- Pause with `P`, quit with `Q`
- Colored terminal rendering
- Ghost personalities: Blinky, Pinky, Inky, and Clyde
- Power pellets and edible ghost combos
- Wraparound tunnel
- Bonus fruit
- Level progression
- Persistent top-5 scoreboard in `.pacman_scores.json`

## Run

```bash
python3 pacman_game.py
```

> Works best in a real terminal window with enough space for the game board.
